# Typography

Text-rendering primitives — labels, links, code blocks, keyboard chips, block quotes, bullet lists.

| Component | Summary |
|---|---|
| [FxLabel](FxLabel.md) | Themed label with `LabelVariant` (display / heading / subheading / body / small / mono). |
| [FxLink](FxLink.md) | Text hyperlink with hover state and optional graphic. |
| [FxText](FxText.md) | Body text node with variant control. |
| [FxBlockquote](FxBlockquote.md) | Left-bar block quote for pulled-out prose. |
| [FxKbd](FxKbd.md) | Rendered keyboard-key chip (⌘, K, etc.). |
| [FxCode](FxCode.md) | Inline monospaced code snippet. |
| [FxBulletList](FxBulletList.md) | Ordered or unordered list of text items. |
