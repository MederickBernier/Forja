# Selection

Choice controls — pick one, pick many, pick a number, pick a color.

| Component | Summary |
|---|---|
| [FxCheckBox](FxCheckBox.md) | Styled checkbox with optional indeterminate state. |
| [FxCheckGroup](FxCheckGroup.md) | Multi-select group of checkboxes with observable selected-values list. |
| [FxRadioGroup](FxRadioGroup.md) | Single-select radio group with observable value property. |
| [FxSwitch](FxSwitch.md) | Pill-style on/off toggle switch. |
| [FxSlider](FxSlider.md) | Numeric range slider with optional value label. |
| [FxComboBox](FxComboBox.md) | Styled dropdown combo box. |
| [FxMultiSelect](FxMultiSelect.md) | Multi-select combo with checkable items and N-selected summary. |
| [FxRangeSlider](FxRangeSlider.md) | Two-thumb slider for numeric range selection. |
| [FxRating](FxRating.md) | Star-rating input with hover preview. |
| [FxColorPicker](FxColorPicker.md) | Styled color picker. |
