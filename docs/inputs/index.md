# Inputs

Text-entry components. Every field carries a shared `InputVariant` (`DEFAULT` / `ERROR` / `SUCCESS`) and helper/error label slots.

| Component | Summary |
|---|---|
| [FxTextField](FxTextField.md) | Text input with icons, helper/error text, and validation-state variants. |
| [FxTextArea](FxTextArea.md) | Multi-line text input with auto-resize and validation. |
| [FxPasswordField](FxPasswordField.md) | Masked input with optional reveal-toggle. |
| [FxNumberField](FxNumberField.md) | Numeric input with min/max clamp, steppers, prefix/suffix. |
| [FxSearchField](FxSearchField.md) | Text input preset with search icon and clear button. |
| [FxAutocomplete](FxAutocomplete.md) | Text input with a filtered suggestion popup. |
| [FxTagInput](FxTagInput.md) | Chip-based tag entry with add-on-Enter and remove-with-backspace. |
| [FxOTPInput](FxOTPInput.md) | Row of single-character boxes for one-time codes. |
| [FxMaskedInput](FxMaskedInput.md) | Text input with a fixed input mask (`###-###`, `AAA-####`). |
| [FxCodeEditor](FxCodeEditor.md) | Syntax-highlightable code editor built on RichTextFX. |
| [FxRichTextEditor](FxRichTextEditor.md) | Rich-text editor with bold/italic/underline toolbar. |
| [FxMarkdownEditor](FxMarkdownEditor.md) | Split source + live-rendered markdown preview. |
| [FxJsonEditor](FxJsonEditor.md) | Code editor preset with a JSON syntax highlighter. |
