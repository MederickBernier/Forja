# Date & Time

Date, time, range, and calendar pickers.

| Component | Summary |
|---|---|
| [FxDatePicker](FxDatePicker.md) | Styled date input with min/max clamp and configurable format pattern. |
| [FxTimePicker](FxTimePicker.md) | Hour + minute (optional seconds) spinner picker with 12/24-hour mode. |
| [FxDateTimePicker](FxDateTimePicker.md) | Composite `FxDatePicker` + `FxTimePicker` producing a `LocalDateTime`. |
| [FxDateRangePicker](FxDateRangePicker.md) | Two-date range with automatic end ≥ start clamp. |
| [FxCalendar](FxCalendar.md) | Month-grid calendar with prev/next navigation and single-day selection. |
| [FxMiniCalendar](FxMiniCalendar.md) | Compact calendar variant for sidebars and popovers. |
