# Validation

Declarative form binding and a rule engine. `FxValidator` is pure logic — no scene dependencies — and can be reused outside of a form.

| Component | Summary |
|---|---|
| [FxValidator](FxValidator.md) | Sequential rule engine with built-in `required` / `email` / `range` / `matches` factories. |
| [FxFormField](FxFormField.md) | Label + control + error slot with `:invalid` / `:required` pseudo-classes. |
| [FxErrorSummary](FxErrorSummary.md) | Aggregate danger-colored list of validation errors. |
| [FxForm](FxForm.md) | Declarative form container that validates every field in one call. |
