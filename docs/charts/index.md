# Charts

Data visualization. First six wrap `javafx.scene.chart.*`; the last four are Canvas-drawn.

| Component | Summary |
|---|---|
| [FxLineChart](FxLineChart.md) | Category X + number Y line chart wrapper. |
| [FxBarChart](FxBarChart.md) | Category X + number Y bar chart wrapper. |
| [FxAreaChart](FxAreaChart.md) | Category X + number Y area chart wrapper. |
| [FxScatterPlot](FxScatterPlot.md) | Number X + number Y scatter plot wrapper. |
| [FxPieChart](FxPieChart.md) | Styled `PieChart` wrapper. |
| [FxDonutChart](FxDonutChart.md) | Donut variant of `FxPieChart`. |
| [FxSparkline](FxSparkline.md) | Compact inline polyline chart. |
| [FxGauge](FxGauge.md) | Semicircle arc gauge with value label. |
| [FxHeatmap](FxHeatmap.md) | Color-lerped grid heatmap for 2D data. |
| [FxRadarChart](FxRadarChart.md) | Polar polygon-per-series radar chart. |
