# Layout

Structural primitives — arrange other components.

| Component | Summary |
|---|---|
| [FxSeparator](FxSeparator.md) | Horizontal or vertical divider line. |
| [FxSpacer](FxSpacer.md) | Absorbs available space between siblings. |
| [FxStack](FxStack.md) | Vertical flow with spacing. |
| [FxRow](FxRow.md) | Horizontal flow with spacing. |
| [FxContainer](FxContainer.md) | Max-width centered container. |
| [FxSection](FxSection.md) | Titled section wrapper. |
| [FxCard](FxCard.md) | Framed content card with variant, header, and footer slots. |
| [FxGrid](FxGrid.md) | Fluent-builder `GridPane` wrapper. |
| [FxFlex](FxFlex.md) | Wrap-flow layout (`FlowPane`) with gap control. |
| [FxAspectRatio](FxAspectRatio.md) | Constrains a single child to a fixed width:height ratio. |
| [FxScrollArea](FxScrollArea.md) | Styled `ScrollPane` wrapper. |
| [FxSplitView](FxSplitView.md) | Styled resizable `SplitPane`. |
| [FxResizablePane](FxResizablePane.md) | Container with user-draggable edge handle. |
| [FxCollapse](FxCollapse.md) | Single-section collapsible with header + expandable body. |
| [FxAccordion](FxAccordion.md) | Stack of `FxCollapse` sections with optional single-open enforcement. |
| [FxStickyHeader](FxStickyHeader.md) | Pins its header to the top of a host `ScrollPane`. |
| [FxResponsive](FxResponsive.md) | Swaps children based on width breakpoints. |
