# Buttons & Actions

Clickable surfaces that carry out a command. Every button honors the shared `ButtonVariant` enum (`PRIMARY` / `SECONDARY` / `GHOST` / `DANGER`).

| Component | Summary |
|---|---|
| [FxButton](FxButton.md) | Text button with variant, loading state, and action handler. |
| [FxIconButton](FxIconButton.md) | Icon-first button with optional label position. |
| [FxToggleButton](FxToggleButton.md) | Stateful button that stays pressed. |
| [FxButtonGroup](FxButtonGroup.md) | Segmented row of toggle buttons with single-select behavior. |
| [FxCopyButton](FxCopyButton.md) | Copies a value to the clipboard and flashes confirmation. |
| [FxSplitButton](FxSplitButton.md) | Primary click surface plus a chevron menu. |
| [FxMenuButton](FxMenuButton.md) | Icon-only kebab-style menu trigger. |
