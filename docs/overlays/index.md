# Overlays

Anything that renders above the app. Most ride on [`OverlayHost`](../foundation.md#overlay-layer) — a per-scene overlay layer installed on first use.

| Component | Summary |
|---|---|
| [FxTooltip](FxTooltip.md) | Styled `Tooltip` with fluent builder and `install(node, text)` shortcut. |
| [FxDialog](FxDialog.md) | Modal dialog on the overlay layer with title, body, and footer slots. |
| [FxConfirmDialog](FxConfirmDialog.md) | Preset dialog with cancel + confirm buttons and boolean callback. |
| [FxFormDialog](FxFormDialog.md) | Preset dialog with save + cancel and optional canSave gate. |
| [FxPopover](FxPopover.md) | Floating panel anchored to a target node. |
| [FxHoverCard](FxHoverCard.md) | Popover driven by target hover with open/close delays. |
| [FxDrawer](FxDrawer.md) | Side-anchored sliding panel with scrim. |
| [FxBottomSheet](FxBottomSheet.md) | Bottom-anchored slide-up sheet with grabber handle. |
| [FxLightbox](FxLightbox.md) | Full-scrim media viewer with prev/next navigation. |
