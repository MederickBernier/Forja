# Media

Video, audio, image gallery, waveform.

Requires the `javafx-media` module at runtime for video/audio playback. Waveform reads PCM via `javax.sound.sampled` (WAV/AU/AIFF supported without extra codecs).

| Component | Summary |
|---|---|
| [FxVideoPlayer](FxVideoPlayer.md) | Video player with play/pause, seek slider, and elapsed/duration label. |
| [FxAudioPlayer](FxAudioPlayer.md) | Compact audio player with play/pause and progress. |
| [FxImageGallery](FxImageGallery.md) | Thumbnail grid that opens `FxLightbox` on click. |
| [FxWaveform](FxWaveform.md) | Canvas-drawn PCM waveform display. |
