# Feedback & Status

User-facing state signals. Most components accept a `SemanticVariant` (`INFO` / `SUCCESS` / `WARNING` / `DANGER` / `ACCENT` / `MUTED` / `DEFAULT`).

| Component | Summary |
|---|---|
| [FxBadge](FxBadge.md) | Small labeled pill for counts and status tags. |
| [FxChip](FxChip.md) | Removable labeled chip. |
| [FxStatusDot](FxStatusDot.md) | Colored dot for at-a-glance status. |
| [FxProgressBar](FxProgressBar.md) | Horizontal progress bar with determinate + indeterminate modes. |
| [FxProgressCircle](FxProgressCircle.md) | Circular progress indicator. |
| [FxSkeleton](FxSkeleton.md) | Shimmering placeholder shape for loading states. |
| [FxAlert](FxAlert.md) | Inline colored notice with icon and dismiss action. |
| [FxBanner](FxBanner.md) | Full-width top-of-page notice with CTA and dismiss. |
| [FxToast](FxToast.md) | Transient bottom-right notification on the overlay layer. |
| [FxNotificationCenter](FxNotificationCenter.md) | Stack of dismissable notifications anchored to a scene corner. |
| [FxEmptyState](FxEmptyState.md) | Centered "nothing here" panel with icon, heading, action. |
| [FxErrorState](FxErrorState.md) | Error-flavored empty state with danger tint. |
| [FxResultPage](FxResultPage.md) | Full-page success / failure / pending / warning summary. |
