# FxCode

Inline monospaced code snippet — a chip with a JetBrains Mono font.

## Usage

```java
FxCode ref = FxCode.builder().text("Forja.install(scene)").build();
```

## Builder methods

| Method | Type | Default | Description |
|---|---|---|---|
| `text(String)` | `String` | `""` | Code text. |
