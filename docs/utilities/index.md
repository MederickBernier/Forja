# Utilities

Cross-cutting helpers that don't fit in a category.

| Component | Summary |
|---|---|
| [FxIcon](FxIcon.md) | Styled icon glyph built on Ikonli (bundles the Feather pack). |
| [FxThemeToggle](FxThemeToggle.md) | Button that toggles a scene between light and dark themes. |
| [FxKeybindingHint](FxKeybindingHint.md) | Row of `FxKbd` chips separated by `+` (e.g. ⌘ + K). |
| [FxSearchHighlight](FxSearchHighlight.md) | `TextFlow` that bolds occurrences of a query in a source string. |
| [FxScrollSpy](FxScrollSpy.md) | Tracks which registered section is visible inside a `ScrollPane`. |
| [FxPortal](FxPortal.md) | Renders a node into the overlay layer on demand. |
| [FxFocusTrap](FxFocusTrap.md) | Cycles keyboard focus among a subtree's descendants. |
